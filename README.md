## üöÄ Quick Start Protocol

### 1. Clone the Sacred Codebase

```bash
git clone <repository-url>
cd pelojams
```

### 2. Install Cosmic Dependencies

```bash

```

Copy `.env.example` to `.env` and add your sacred tokens:


